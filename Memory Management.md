# Memory Concepts in Java:

Memory Management is all about `Garbage Collection` i.e. `GC` in java  

`GC` runs on Heap memory.  

`Heap Memory` is bifurcated into following memory segments:  
1. Young Generation
   - Eden Space
   - Survivor Space 1
   - Survivor Space 2
2. Old Generation
